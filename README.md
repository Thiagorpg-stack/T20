🐉 Calculadora de Ameaças - Tormenta20
Uma ferramenta web rápida, offline e interativa para criação de ameaças (monstros e NPCs) para o sistema de RPG Tormenta20.

Desenvolvida para ajudar Mestres de Jogo a consultarem instantaneamente os atributos de combate e resistências de criaturas com base em seu Nível de Desafio (ND), sem precisar folhear o manual durante a sessão.

✨ Funcionalidades
Alterne instantaneamente entre os três tipos de ameaças oficiais do sistema:

⚔️ Solos: Feitas para enfrentar o grupo de heróis sozinhas.

👥 Lacaios: Criaturas mais fracas, feitas para lutar em bando.

🌟 Especiais: O meio-termo ideal para pequenos grupos de inimigos.

Banco de Dados Oficial: A calculadora não usa aproximações matemáticas. Ela utiliza os dados das tabelas de criação de ameaças do livro de Tormenta20.

Totalmente Offline: Funciona no computador, tablet ou celular sem depender de conexão com a internet.

Dicas de Design Integradas: Exibe recomendações contextuais da quantidade de habilidades sugeridas por patamar de desafio para facilitar a criação.

🚀 Como Usar
utilizá-la é rendondamente simples:

Faça o download do arquivo.

Dê um duplo clique no arquivo para abri-lo em qualquer navegador web (Chrome, Edge, Safari, Firefox, etc.).

Na interface:

Escolha o Tipo de Ameaça (Solo, Lacaio ou Especial).

Selecione o Nível de Desafio (ND) desejado (de 1/4 até S+).

Todos os atributos (PV, Defesa, Ataque, Dano, Resistências e CD) serão atualizados instantaneamente na tela.
